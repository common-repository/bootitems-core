== Description ==

Bootitems Core is a companion plugin for Bootitems Themes, which provides core functionality and extends free themes features by adding functionality to import demo data content in just a click. The plugin requires the [Advanced Import : One Click Import for WordPress or Theme Demo Data](https://wordpress.org/plugins/advanced-import/) plugin to be installed.

How to use this Bootitems Core?

1. While installing free themes by bootitems do kindly install & activate the plugin "Bootitems Core
2. The plugin requires the [Advanced Import : One Click Import for WordPress or Theme Demo Data](https://wordpress.org/plugins/advanced-import/) plugin to be installed. So install and active it.
2. To import demo contents, please go to Appearance > Demo Import.
3. Choose your demo by checking live preview and then click import button to import the demo.
5. Feel free to contact bootitems support team at ( https://bootitems.com/contact/ ) if you need any assistance.


== Installation ==

You can install Bootitems Core Plugins as following two ways:
1. Upload zip file from Dashboard->Plugin->Add New "upload plugin".
2. Extract Bootitems Core and placed it to the "/wp-content/plugins/" directory.
    - Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==

= Is Bootitems Core is a free plugin? =

Yes, it is a free plugin.

= Does this plugin works with any WordPress theme =

No, This plugin will only work with the themes developed by [Bootitems](https://bootitems.com/).
